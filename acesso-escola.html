AcessoEscola Acessibilidade para todos

Bem-vindo ao AcessoEscola

Uma ferramenta desenvolvida para tornar o ambiente escolar mais acessível para estudantes com deficiência visual ou baixa visão.

Recursos Principais

Leitura em Voz Alta

Ouça qualquer texto com síntese de voz em português.

Alto Contraste

Modo de alto contraste para facilitar a leitura e navegação.

Tamanho do Texto

Ajuste o tamanho da fonte para sua melhor visualização.

Navegação Simplificada

Interface intuitiva com suporte completo a teclado e leitores de tela.

Dicas de Uso

  • Use a tecla Tab para navegar entre os elementos
  • Pressione Enter ou Espaço para ativar botões
  • Pressione Esc para fechar janelas
  • Use o menu inferior para acessar todas as seções
  • Ative o Alto Contraste no botão superior direito

Compatibilidade com Leitores de Tela

Este aplicativo é totalmente compatível com leitores de tela como NVDA, JAWS e VoiceOver. Todos os elementos possuem descrições adequadas para navegação assistiva.
